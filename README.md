Welcome to Blogie Blog. It's a Blog Web Application.It does few things which you must not miss to checkout. 
(Note: For now its only the Backend part made with nodejs, expressjs , ejs.) 
1. Post Creation: Users will be able to create new posts.

2. Post Viewing: The Blog page allows the user to view all their posts.

3. Post Update/Delete: Users will be able to edit and delete posts as needed.

4. Styling: The application is well-styled and responsive,ensuring a good user experience on both desktop and mobile devices.
